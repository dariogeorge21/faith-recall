# Saint Images

This directory should contain 10 PNG images of saints:

1. `peter.png` - Saint Peter
2. `sebastian.png` - Saint Sebastian
3. `francis.png` - Saint Francis of Assisi
4. `therese.png` - Saint Thérèse of Lisieux
5. `matthew.png` - Saint Matthew
6. `mark.png` - Saint Mark
7. `luke.png` - Saint Luke
8. `john.png` - Saint John the Evangelist
9. `paul.png` - Saint Paul
10. `joseph.png` - Saint Joseph

**Image Requirements:**
- Format: PNG
- Recommended size: 300x400px or similar portrait aspect ratio
- Clear, recognizable representations of each saint
- High quality, suitable for display on cards

For development/testing, you can use placeholder images or download appropriate saint images from public domain sources.
